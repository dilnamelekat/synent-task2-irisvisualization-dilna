# Iris Data Visualization Project

## Problem Statement

The objective of this project is to analyze and visualize the Iris dataset using Python to understand the characteristics of different flower species.

## Dataset Details

* Dataset: Iris Dataset
* Total Records: 150
* Species:

  * Iris-setosa
  * Iris-versicolor
  * Iris-virginica

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn

## Approach

### Data Loading

Loaded the Iris dataset using Pandas.

### Data Exploration

* Checked dataset information using `info()`
* Generated summary statistics using `describe()`
* Counted the number of samples in each species

### Visualizations Created

#### Histogram

Visualized the distribution of numerical features.

#### Scatter Plot

Compared Petal Length and Petal Width for different species.

#### Bar Chart

Compared average measurements across flower species.

## Key Insights

* The dataset contains 150 flower samples.
* Each species contains 50 samples.
* Iris-setosa is clearly different from the other two species.
* Petal Length and Petal Width help distinguish flower species.
* Iris-virginica generally has larger petal measurements.

## Conclusion

Data visualization helped identify patterns and differences between flower species. These insights can be useful for future machine learning classification tasks.
